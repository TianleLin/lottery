# lottery:字学镜像计划·抽奖转盘·Web端

## 项目简介

使用VUE和轻服务实现一个6个奖项，可以点击转动抽奖的抽奖转盘，转盘历经5秒转速逐渐降低直至停止，抽出奖品，包含Web端和Server端。此仓库为Web端。


[客户端地址]: https://qciwmk.web.cloudendpoint.cn/
## 用户需求·Web

- 转盘中有6个奖品
- 用户点击中间抽奖按钮可以进行抽奖
- 5秒后抽奖结束
- 转盘的转速逐渐降低
- 弹出中奖信息
